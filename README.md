# HRMS Lite – Full Stack Application

## 📌 Project Overview

HRMS Lite is a lightweight Human Resource Management System built as a full-stack web application.  
It allows an admin to manage employee records and track daily attendance.

The application demonstrates end-to-end full-stack development including:

- RESTful API design
- Database integration (SQL)
- Frontend–Backend integration
- Error handling & validations
- Production deployment

This project simulates a basic internal HR tool with a clean and professional UI.
